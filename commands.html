Bot Commands

Commands Are Listed Below 

    /play (To play the music)
    /pause (To pause the music)
    /resume (To resume the music)
    /queue (To check song queue)
    /nowplaying (To check currently playing song)
    /skip (To skip the track)
    /help (For help)
    /info (For Information about developers and support server)
    /servers (To check how much servers this bot is in)
    /ping (To check current bot ping)
    /stop (To stop listening)
    /24-7 (To make bot 24/7 in the server)

We will add or can remove any commands from the list in future, depends on the workings. So these will be our default commands! Join support server for more information.
